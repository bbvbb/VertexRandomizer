名称：VertexRandomizer
+---------------------------------------------------------------------------+

技术支持：沐风课堂（http://www.bbvbb.com）

功能介绍：
VertexRandomizer，是一款由#沐风课堂#用MAXScript脚本语言开发的3dsMax小工具，可以对一个或多个多边形对象顶点进行随机移动。移动可以发生在三维空间任意方向，也可以约束在“边”、“面”、“法线”上移动。
